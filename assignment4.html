Building 3D models and vectors in OnShape

I leveraged Joshua's in-class tutorial and the video that Krista so graciously recorded and uploaded as my primary reference point. While Rhino had a lot of tutorials to show you how to do things, there weren't as many for OnShape. I think there may be an expectation that it's so easy to use that a tutorial isn't really needed or it just isn't as prolific. Either way, I spent a lot of time clicking on random things and using the Dimension button to try to force things to be the right shape.

I ended up 3D printing a pulley for my handle since I wasn't completely sure how far I would get and I wanted options for my motion.

04

Export to Illustrator

Exporting to Illustrator is pretty straightforward with a key option to pay attention to when you open the file in Illustrator to make sure you don't lose any of your measurements.

  • Select the layer that you'd like to export
  • Right click it
  • Change the file name and select the DXF format
  • Click Export
  • Open the file in Illustrator
  • Select "Scale by:"
  • Change the dropdown that says "Inches" to "Millimeters"
  • Change "Unit(s)" to 1 since the original 25.4 is how many millimeters equals 1 inch


  • Change the stroke width to .001 to prepare for vector cutting

I exported each folder into its own AI file to make cutting easier.

03

Assembly Execution Details

Assembly was tricky. A few things I realized at 2AM

  • The pole should NOT come down to a point. It should have had a wide base. I ended up cutting off the point with scissors, cutting one of the excess hexagons into each of its sides, and hot glueing it to the end.




  • The hole around the pole should leave enough space to let the poles smoothly move up and down. I ended up smooshing the corrugated cardboard so that it could do this more seamlessly
  • Hot glue is your best friend.
    • Hot glue... the cams together to create a consistent shape
    • Hot glue... the tips of the structure so the cardboard is more stable and snaps into place
    • Hot glue... a broader base to the poles so it doesn't just tip over
    • Hot glue... all the scene pieces to the hexagon and the hexagon to the poles
    • Hot glue... your fingers on accident to remind you that you're alive

Once I figured that out, it was more smooth sailing

And we're done
