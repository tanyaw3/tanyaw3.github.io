Building Vectors in Grasshopper + Rhino

I've never used Grasshopper, Rhino, or anything like Grasshopper or Rhino, so I felt completely out of my comfort zone. I probably should have done the same as assignment 1 but thought it would be worthwhile to try something a little more complex to understand the program better. It took me hours to just understand what I was doing and I referenced a lot of video's including Nadya's video and many, many, YouTube videos.

I started by building the scene. After spending 1.5 days in Rhino and Grasshopper, Rhino wouldn't save my file and completely crashed. If not for saving most of my files in Grasshopper, I would have lost all of my work.

Layered Scene in Rhino

Layered Scene in Grasshopper

I'm still not sure if I did my work the right way or not, but I've included some of my learnings and discoveries below.

  • Use a mouse. It's so much faster.
  • Plan out your layers beforehand so you can bake to them.
  • Copy and paste is your best friend. I would build a component once, like number slider and reuse it
  • Don't bake shapes until the very end
  • Just stay in millimeter units
  • Build everything together before shifting it to different layers
  • Always make the width of the space for the attach points smaller than the caliper measurement

How to Make Curves

  • Type "Point"
  • Set "Multiple Points"
  • Type "Nurbs Curve"
  • Type "Number Slider"

How to Move Objects

  • Type "Move"
  • Connect "Geometry"
  • Type "Vector XYZ"
  • Get two "Number Slider" selections

How to Duplicate items via "linear array"

How to Merge multiple shapes to make duplication easier

  • Type "Merge"
  • Connect all the geometries together
  • Type "Vector XYZ"
  • Get two "Number Slider" selections

04

Export to Illustrator

Fortunately, exporting from Rhino to Illustrator was pretty straightforward.

  • Go to File
  • Click Export
  • Select Adobe Illustrator from the drop-down menu
  • Click Export
  • Set Export Options-- I wanted it to maintain scale

    • When I opened the file in Illustrator, I made a few edits to make it print ready:

    • Change the stroke to .001pt, so it'll print as a vector file
    • This caused my curves to render a little strangely
    • To solve this, I used the Direct Selection Tool, selected the different anchor points, and moved it around to solve the issue
    • Then, I moved the vectors to different art boards for printing. I knew I was lasercutting on 8.5"x11" paper, so I split it across three boards
    • Since the stand pieces would be lasercut on cardboard, I put that on a separate file
